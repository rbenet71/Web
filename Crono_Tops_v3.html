Ayuda de Crono Tops

Crono Tops es una aplicación para tomar tiempos en carreras de bicicletas. Puedes crear múltiples cronómetros para diferentes puntos de control.

Cómo usar:

  • Nuevo Crono: Crea un nuevo cronómetro con una pestaña.
  • Nombre del Top: Asigna un nombre al punto de control (ej: "Frente Carrefour Prat").
  • Botón Inicio/Top: Al pulsar por primera vez se inicia el cronómetro. Las siguientes pulsaciones registran tiempos (tops).
  • Dorsal: Asigna el número de dorsal a cada tiempo registrado.
  • Eliminar Top: Elimina un tiempo registrado si es necesario.

Los tiempos se muestran en formato minutos:segundos.décimas y la diferencia se calcula automáticamente entre cada top registrado.
