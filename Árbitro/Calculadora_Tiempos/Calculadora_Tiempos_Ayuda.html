Descripción General

La Calculadora de Tiempos para Ciclismo es una herramienta especializada diseñada específicamente para árbitros, comisarios y oficiales de carreras ciclistas. Permite realizar cálculos precisos de tiempos, promedios, vueltas y operaciones específicas necesarias durante el desarrollo de competiciones ciclistas.

Funcionalidades Principales

Modos de Operación

  • Modo Decimal: Funciona como una calculadora convencional para cálculos matemáticos básicos
  • Modo Tiempo: Especializado para trabajar con horas, minutos y segundos en formato natural

Operaciones Específicas para Carreras

  • Suma y Resta de Tiempos: Para calcular tiempos parciales y totales
  • Cálculo de Promedio: Velocidad media en km/h (ideal para controles de tiempo)
  • Número de Vueltas: Calcula vueltas completas en función del tiempo por vuelta
  • Incrementos Porcentuales: Para cálculos de tiempos límite y márgenes

Funciones de Memoria Avanzadas

  • M+: Suma el valor actual a la memoria (conserva formato tiempo/decimal)
  • MR: Recupera el valor almacenado en memoria
  • MC: Limpia completamente la memoria

Herramientas para Árbitros

  • Historial de operaciones: Guarda los últimos 3 cálculos
  • Multilenguaje: Español, Catalán e Inglés
  • Interfaz optimizada: Diseñada para uso rápido en competiciones
  • Persistencia: Guarda preferencias de idioma

Otras Funciones Disponibles

Sistema de Sugerencias Integrado

Acceso: Botón "Actualizaciones/Sugerencias"

Funcionalidad: Permite enviar sugerencias, reportar errores o proponer mejoras directamente desde la aplicación. Los datos se envían a un formulario de Google Forms para su evaluación.

Campos requeridos: Email y Ciudad (para segmentación geográfica de sugerencias)

Rendimiento mejorado: Carga más rápida y experiencia de usuario fluida

Cómo Usar en Competiciones

1 Cálculo de Tiempos Parciales

Uso: En controles de paso, suma los tiempos de diferentes corredores o equipos

Ejemplo: Tiempo líder + diferencia = tiempo de cada corredor

2 Cálculo de Velocidad Media

Uso: Verificar si los corredores mantienen el ritmo establecido

Ejemplo: Distancia 25km, Tiempo 35m 42s → Velocidad media: 42.15 km/h

3 Cálculo de Vueltas en Circuitos

Uso: Determinar cuántas vueltas completarán los corredores en el tiempo disponible

Ejemplo: Tiempo por vuelta: 4m 15s, Tiempo carrera: 2h → 28 vueltas completas

4 Márgenes de Tiempo

Uso: Calcular tiempos límite y márgenes permitidos

Ejemplo: Tiempo referencia + 10% = tiempo máximo permitido

5 Uso de Memoria en Cálculos Complejos

Uso: Almacenar tiempos intermedios durante cálculos de etapas multi-segmento

Ejemplo: Guardar tiempo de ascenso, luego sumar tiempo de descenso

  • Funcionamiento offline: Una vez instalada, no requiere internet
  • Actualizaciones automáticas: Detecta e instala nuevas versiones

  • Alto rendimiento: Optimizada para uso rápido en competiciones
  • Diseño responsive: Adaptable a móviles, tablets y ordenadores
  • Sistema de feedback: Canal directo para sugerencias de árbitros

Consejos para Árbitros

  • Antes de la carrera: Instala la app en tu móvil para uso offline
  • Durante la carrera: Usa la memoria para cálculos complejos
  • Para controles de tiempo: Utiliza la función de promedio
  • En circuitos: Configura el cálculo de vueltas al inicio
  • Comunicación: Usa el mismo idioma que el panel de árbitros
  • Verificación: Siempre verifica cálculos importantes 2 veces

General Description

The Time Calculator for Cycling is a specialized tool designed specifically for referees, commissioners and race officials. It allows precise calculations of times, averages, laps and specific operations needed during cycling competitions.

Main Features

Operation Modes

  • Decimal Mode: Works as a conventional calculator for basic mathematical calculations
  • Time Mode: Specialized for working with hours, minutes and seconds in natural format

Race-Specific Operations

  • Time Addition/Subtraction: For calculating partial and total times
  • Average Calculation: Average speed in km/h (ideal for time controls)
  • Number of Laps: Calculates complete laps based on lap time
  • Percentage Increases: For time limit and margin calculations

Advanced Memory Functions

  • M+: Adds current value to memory (preserves time/decimal format)
  • MR: Recalls value stored in memory
  • MC: Completely clears memory

Tools for Referees

  • Operation history: Saves last 3 calculations
  • Multilingual: Spanish, Catalan and English
  • Optimized interface: Designed for fast use in competitions
  • Persistence: Saves language preferences

Other Available Features

Integrated Suggestions System

Access: "Updates/Suggestions" button

Functionality: Allows sending suggestions, reporting errors or proposing improvements directly from the application. Data is sent to a Google Form for evaluation.

Required fields: Email and City (for geographical segmentation of suggestions)

Advanced Installation System

Access: "Install App" button

Functionality: Step-by-step guide to install the application as PWA (Progressive Web App) on different devices:

  • iOS: Specific instructions for iPhone/iPad
  • Android: Automatic installation as PWA
  • Windows: .url shortcut download

Offline Operation

Advantages: Once installed, the application works without internet connection

Automatic updates: The system detects and installs new versions

Improved performance: Faster loading and smooth user experience

How to Use in Competitions

1 Partial Time Calculation

Use: At checkpoints, add times of different riders or teams

Example: Leader time + difference = each rider's time

2 Average Speed Calculation

Use: Verify if riders maintain established pace

Example: Distance 25km, Time 35m 42s → Average speed: 42.15 km/h

3 Circuit Lap Calculation

Use: Determine how many laps riders will complete in available time

Example: Lap time: 4m 15s, Race time: 2h → 28 complete laps

4 Time Margins

Use: Calculate time limits and allowed margins

Example: Reference time + 10% = maximum allowed time

5 Memory Use in Complex Calculations

Use: Store intermediate times during multi-segment stage calculations

Example: Save climb time, then add descent time

Technical Features

  • Progressive Web App (PWA): Installable as native application
  • Offline operation: Once installed, doesn't require internet
  • Automatic updates: Detects and installs new versions
  • Cross-platform: Works on iOS, Android, Windows, macOS
  • High performance: Optimized for fast use in competitions
  • Responsive design: Adaptable to mobiles, tablets and computers
  • Feedback system: Direct channel for referee suggestions

Tips for Referees

  • Before race: Install app on your mobile for offline use
  • During race: Use memory for complex calculations
  • For time controls: Use average function
  • In circuits: Configure lap calculation at start
  • Communication: Use same language as referee panel
  • Verification: Always verify important calculations twice

  3. Scroll and select "Add to Home Screen"
  4. Customize name and tap "Add"

For Android:

  1. Open Chrome and navigate to calculator
  2. Tap three-dot menu (upper right corner)
  3. Select "Install app" or "Add to Home Screen"
  4. Follow browser instructions
  5. App will install automatically

Installation advantages:

  • Immediate access: Direct icon on home screen
  • Works without internet: Essential in areas without coverage
  • Automatic updates: You'll always have latest version
  • Higher speed: Instant loading as native app
  • Notifications: Receive alerts about new features
