¿Qué es Crono Cuenta Atrás?

Crono Cuenta Atrás es una aplicación profesional para controlar las salidas en carreras deportivas. Proporciona una cuenta atrás visual y sonora para organizar salidas escalonadas de corredores o participantes.

Configuraciones de hardware recomendadas

La aplicación está diseñada para funcionar en múltiples dispositivos, permitiendo flexibilidad según tus necesidades:

Portátil

Para el control principal

Tablet

Para visualización secundaria

Móvil

Para control portátil

Pantalla TV

Para visualización pública

Altavoz Bluetooth

Para audio amplificado

Ideas de configuración

  • Configuración básica: Usa un portátil conectado a una pantalla grande de TV
  • Configuración móvil: Tableta + altavoz Bluetooth para máxima portabilidad
  • Configuración doble: Portátil para control + tableta para visualización en línea de salida
  • Configuración profesional: Todos los dispositivos conectados y sincronizados

Tipos de carreras compatibles

Descensos

Salidas escalonadas en descenso de montaña

Contrarreloj

Salidas individuales con intervalos

Salidas masivas

Grupos que salen con intervalos

Otras disciplinas

Cualquier deporte con salidas cronometradas

Cómo usar la aplicación

1 Crear o seleccionar una carrera
  • Crea una nueva carrera con nombre y descripción
  • Selecciona una carrera existente para continuar donde la dejaste
  • Los datos se guardan automáticamente
2 Configurar la cadencia de salida

Siempre igual

Todos los corredores salen con el mismo intervalo de tiempo

Ejemplo: Cada corredor sale cada 15 segundos

Varios tramos

Configura diferentes intervalos para grupos de corredores

Ejemplo: Del 1 al 10: cada 10s, del 11 al 50: cada 20s

Ejemplo práctico de configuración por tramos

Para una carrera con 60 participantes:

Tramo 1: Dorsales 1-3 → Cada 10 segundos

Tramo 2: Dorsales 4-20 → Cada 20 segundos

Tramo 3: Dorsales 21-60 → Cada 15 segundos

Esta configuración permite mayor densidad al principio y al final, con un intervalo más amplio en el centro para evitar aglomeraciones.

3 Configurar audio
  • Altavoz Bluetooth: Conecta un altavoz para que el sonido llegue a toda la línea de salida
  • Pitidos electrónicos: Sonidos precisos para cada segundo
  • Voz grabada: Conteo vocal "10, 5, 4, 3, 2, 1, ¡SALIDA!"
  • Sin sonido: Solo indicadores visuales para entornos ruidosos
4 Iniciar la cuenta atrás
  1. Define desde qué corredor comenzar
  2. Haz clic en "INICIAR CUENTA ATRÁS"
  3. La pantalla cambiará al modo de ejecución
  4. ¡Todo está listo para comenzar!

Listado de salidas y exportación

Funcionalidades avanzadas

  • Listado completo: Al finalizar la carrera, obtén un listado detallado de todas las salidas
  • Añadir dorsales: Puedes asignar números de dorsal a cada posición de salida
  • Anotaciones personalizadas: Añade observaciones específicas para cada corredor
  • Exportación a Excel: Exporta el listado completo a formato CSV/Excel para su procesamiento
  • Uso posterior: Los datos exportados pueden usarse para:
    • Cálculo de tiempos finales
    • Generación de clasificaciones
    • Estadísticas de carrera
    • Informes oficiales

Solución de problemas

Problemas comunes

  • No se escuchan los sonidos:
    • Asegúrate de que el volumen del navegador está activado
    • Permite la reproducción de audio en el navegador
    • Verifica la conexión Bluetooth si usas altavoz externo
  • La pantalla se apaga: La aplicación incluye un sistema para mantener la pantalla activa
  • Los datos no se guardan: Los datos se guardan automáticamente en tu navegador (LocalStorage)
  • La cuenta atrás no es precisa: Usa la función de sincronización en el menú de configuración
  • Problemas de conectividad Bluetooth:
    • Empareja el altavoz antes de abrir la aplicación
    • Asegúrate de que el dispositivo está dentro del rango
    • Reinicia la conexión si hay problemas de audio

Instalación como App

Para iPhone/iOS:

  1. Abrir Safari y navegar a la aplicación
  2. Tocar el icono de compartir (cuadrado con flecha hacia arriba)
  3. Desplazarse y seleccionar "Añadir a pantalla de inicio"
  4. Personalizar el nombre y tocar "Añadir"
  5. La app aparecerá en la pantalla de inicio

Para Android:

  1. Abrir Chrome y navegar a la aplicación
  2. Tocar el menú de tres puntos (esquina superior derecha)
  3. Seleccionar "Instalar app" o "Añadir a pantalla de inicio"
  4. Seguir las instrucciones del navegador
  5. La app se instalará automáticamente

Ventajas de la instalación:

  • Acceso inmediato: Icono directo en pantalla de inicio
  • Funciona sin internet: Esencial en zonas sin cobertura
  • Actualizaciones automáticas: Siempre tendrás la última versión
  • Mayor velocidad: Carga instantánea como app nativa
  • Pantalla completa: Sin barras del navegador

What is Countdown Timer?

Countdown Timer is a professional application to control race starts in sporting events. It provides a visual and audible countdown to organize staggered starts of runners or participants.

Recommended hardware configurations

The application is designed to work on multiple devices, allowing flexibility according to your needs:

Laptop

For main control

Tablet

For secondary display

Mobile

For portable control

TV Screen

For public display

Bluetooth Speaker

For amplified audio

Setup ideas

  • Basic setup: Use a laptop connected to a large TV screen
  • Mobile setup: Tablet + Bluetooth speaker for maximum portability
  • Dual setup: Laptop for control + tablet for start line display
  • Professional setup: All devices connected and synchronized

Supported race types

Downhill

Staggered starts in mountain downhill

Time Trial

Individual starts with intervals

Mass starts

Groups starting with intervals

Other disciplines

Any sport with timed starts

How to use the application

1 Create or select a race
  • Create a new race with name and description
  • Select an existing race to continue where you left off
  • Data is saved automatically
2 Configure start cadence

Always the same

All racers start with the same time interval

Example: Each racer starts every 15 seconds

Multiple intervals

Configure different intervals for groups of racers

Example: From 1 to 10: every 10s, from 11 to 50: every 20s

Practical example of section configuration

For a race with 60 participants:

Section 1: Numbers 1-3 → Every 10 seconds

Section 2: Numbers 4-20 → Every 20 seconds

Section 3: Numbers 21-60 → Every 15 seconds

This configuration allows higher density at the beginning and end, with a wider interval in the center to avoid congestion.

3 Configure audio
  • Bluetooth speaker: Connect a speaker so sound reaches the entire start line
  • Electronic beeps: Precise sounds for each second
  • Recorded voice: Vocal countdown "10, 5, 4, 3, 2, 1, GO!"
  • No sound: Only visual indicators for noisy environments
4 Start the countdown
  1. Define from which racer to start
  2. Click "START COUNTDOWN"
  3. The screen will change to execution mode
  4. Everything is ready to go!

Start list and export

Advanced features

  • Complete list: When the race finishes, get a detailed list of all starts
  • Add race numbers: You can assign race numbers to each start position
  • Custom annotations: Add specific observations for each rider
  • Export to Excel: Export the complete list to CSV/Excel format for processing
  • Post-use: Exported data can be used for:
    • Final time calculation
    • Classification generation
    • Race statistics
    • Official reports

Troubleshooting

Common problems

  • No sound is heard:
    • Make sure browser volume is on
    • Allow audio playback in the browser
    • Check Bluetooth connection if using external speaker
  • Screen turns off: The application includes a system to keep the screen active
  • Data is not saved: Data is automatically saved in your browser (LocalStorage)
  • Countdown is not accurate: Use the sync function in the configuration menu
  • Bluetooth connectivity issues:
    • Pair the speaker before opening the application
    • Make sure the device is within range
    • Restart connection if there are audio problems

Qu'est-ce que Compte à Rebours?

Compte à Rebours est une application professionnelle pour contrôler les départs de course dans les événements sportifs. Il fournit un compte à rebours visuel et sonore pour organiser des départs échelonnés de coureurs ou participants.

  • Configuration mobile: Tablette + haut-parleur Bluetooth pour une portabilité maximale

  • Configuration professionnelle: Tous les appareils connectés et synchronisés
