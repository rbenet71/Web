# Changelog - VideoGest

Todos los cambios notables en esta aplicación se documentarán en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-18
### Added
- Versión inicial de VideoGest
- Funcionalidades básicas:
  - Reducción de tamaño de video
  - Corte de video
  - Conversión de formato
  - Reversión de video
  - Extracción de frames a JPG
  - Unión de videos
- Soporte multilingüe (ES, CA, EN, FR)
- Sistema de almacenamiento persistente
- PWA completa con Service Worker
- Instalación como aplicación nativa
- Sistema de importación/exportación de configuraciones
- Interfaz responsiva y moderna

### Technical
- Arquitectura modular con clases ES6
- Cache inteligente con Service Worker
- Sistema de traducciones dinámicas
- Persistencia con localStorage
- Compatibilidad con FFMPEG
- Offline-first design
