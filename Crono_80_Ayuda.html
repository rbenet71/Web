Nota del Autor

Esta aplicación la ha desarrollado Roberto Benet, comisario del comité catalán, para que cualquier árbitro la pueda utilizar en carreras de ciclismo de forma gratuita. Estoy abierto a cualquier tipo de mejora o implementación de nuevas funciones.

Descripción General

El Cronómetro Control 80% es una herramienta profesional diseñada para carreras de BTT y Ciclo-cross que implementa la regla del 80% para determinar cuándo un corredor debe ser retirado de la competición.

Funcionalidades Principales

Gestión de Carreras

  • Nueva Carrera: Crear una nueva carrera con nombre personalizado
  • Seleccionar Carrera: Cambiar entre diferentes carreras guardadas
  • Eliminar Carrera: Borrar una carrera existente
  • Persistencia Automática: Los datos se guardan automáticamente

Control del Cronómetro

  • Iniciar/Parar: Controlar el cronómetro principal con opciones de reanudación
  • Fijar Tiempo: Establecer manualmente la hora de salida o el tiempo actual
  • Reiniciar: Restablecer completamente el cronómetro y todos los datos
  • Historial de Horas: Acceso rápido a horas de inicio anteriores

Registro de Vueltas

  • Paso Primer Corredor: Registrar el tiempo de cada vuelta del líder
  • Vuelta Actual: Muestra el número de vuelta en curso
  • Edición de Tiempos: Modificar tiempos de vueltas registradas
  • Eliminar Vueltas: Borrar la última vuelta registrada

Control de Corredores

  • Retirar Corredor: Marcar corredores como retirados en la vuelta actual
  • DNF-Retirado: Marcar corredores como No Finalizaron (vuelta 0)
  • Edición de Retirados: Modificar datos de corredores retirados
  • Ordenación Automática: Los retirados se ordenan por vuelta y tiempo

TRES FORMAS DE PONER EN MARCHA EL CRONÓMETRO

1. Pulsando "Iniciar Cronómetro"

Cuando usar: Para una carrera que comienza en este momento.

Proceso: Simplemente pulsa el botón "Iniciar Cronómetro" cuando dé la salida la carrera. El sistema registrará automáticamente la hora actual como hora de inicio.

Ventaja: La forma más rápida y sencilla de empezar.

2. Fijando el Tiempo del Cronómetro

Cuando usar: Cuando necesitas establecer un tiempo específico de cronómetro (ej: 1h 30m 15s).

Proceso: Pulsa "Fijar Tiempo Crono" y selecciona la opción "Por Hora de Crono". Introduce las horas, minutos y segundos deseados.

Ventaja: Útil para reanudar una carrera después de una parada o para establecer tiempos específicos.

3. Fijando la Hora de Inicio

Cuando usar: Cuando conoces la hora exacta de salida de la carrera.

Proceso: Pulsa "Fijar Tiempo Crono" y selecciona "Por Hora de Inicio". Introduce la hora exacta de salida. El sistema calculará automáticamente el tiempo transcurrido.

Ventaja: Perfecto cuando la carrera ya ha comenzado y quieres sincronizar el cronómetro con la hora real de salida.

Extra: Accede al historial de horas de inicio anteriores para seleccionar rápidamente horas usadas en carreras previas.

Cómo Usar

1 Configurar la Carrera

Crear una nueva carrera o seleccionar una existente. Escoger una de las tres formas de iniciar el cronómetro según tus necesidades.

2 Registrar la Primera Vuelta

Presionar "Paso Primer Corredor" cuando el líder complete la primera vuelta. Este tiempo establece la base para el cálculo del 80%.

3 Seguimiento Continuo

Registrar cada vuelta subsiguiente del líder. El sistema calcula automáticamente el tiempo límite para cada vuelta.

4 Control de Retirados

Usar "Retirar Corredor" para corredores que abandonan durante la carrera. Usar "DNF-Retirado" para corredores que no finalizan.

5 Exportar Resultados

Al finalizar, exportar los datos para análisis posterior en formato CSV compatible con Excel.

Regla del 80%

El sistema calcula el tiempo límite para cada vuelta sumando:

  • Tiempo de la última vuelta registrada del líder
  • 80% del tiempo de la primera vuelta

Cuando un corredor supera este tiempo límite, debe ser retirado de la competición.

General Description

The Stopwatch 80% Control is a professional tool designed for MTB and Cyclo-cross races that implements the 80% rule to determine when a rider should be withdrawn from the competition.

Main Features

Race Management

  • New Race: Create a new race with custom name
  • Select Race: Switch between different saved races
  • Delete Race: Delete an existing race
  • Automatic Persistence: Data is automatically saved

Stopwatch Control

  • Start/Stop: Control main stopwatch with resume options
  • Set Time: Manually set start time or current time
  • Reset: Completely reset stopwatch and all data
  • Time History: Quick access to previous start times

Lap Recording

  • First Rider Pass: Record time of each lap by the leader
  • Current Lap: Shows current lap number
  • Time Editing: Modify recorded lap times
  • Delete Laps: Remove last recorded lap

Rider Control

  • Withdraw Rider: Mark riders as withdrawn in current lap
  • DNF-Withdrawn: Mark riders as Did Not Finish (lap 0)
  • Editing Withdrawn: Modify data of withdrawn riders
  • Automatic Sorting: Withdrawn riders sorted by lap and time

THREE WAYS TO START THE STOPWATCH

1. Pressing "Start Stopwatch"

When to use: For a race starting right now.

Process: Simply press the "Start Stopwatch" button when the race starts. The system will automatically record the current time as start time.

Advantage: The fastest and simplest way to start.

2. Setting Stopwatch Time

When to use: When you need to set a specific stopwatch time (eg: 1h 30m 15s).

Process: Press "Set Stopwatch Time" and select "By Stopwatch Time" option. Enter desired hours, minutes and seconds.

Advantage: Useful to resume a race after a stop or to set specific times.

3. Setting Start Time

When to use: When you know the exact race start time.

Process: Press "Set Stopwatch Time" and select "By Start Time". Enter exact start time. System will automatically calculate elapsed time.

Advantage: Perfect when race has already started and you want to sync stopwatch with actual start time.

Extra: Access previous start times history to quickly select times used in previous races.

How to Use

1 Set Up the Race

Create a new race or select an existing one. Choose one of the three ways to start the stopwatch according to your needs.

2 Record First Lap

Press "First Rider Pass" when leader completes first lap. This time sets the base for 80% calculation.

3 Continuous Tracking

Record each subsequent lap by leader. System automatically calculates cutoff time for each lap.

4 Withdrawn Riders Control

Use "Withdraw Rider" for riders who abandon during race. Use "DNF-Withdrawn" for riders who do not finish.

5 Export Results

When finished, export data for later analysis in CSV format compatible with Excel.

80% Rule

The system calculates cutoff time for each lap by adding:

  • Time of the last recorded lap by the leader
  • 80% of the first lap time

When a rider exceeds this cutoff time, they must be withdrawn from the competition.

Règle des 80%

Le système calcule le temps limite pour chaque tour en ajoutant:

  • Temps du dernier tour enregistré du leader
  • 80% du temps du premier tour

Lorsqu'un coureur dépasse ce temps limite, il doit être retiré de la compétition.

Caractéristiques Avancées

  • Édition Avancée: Modifier les temps des tours et des coureurs retirés
  • Langues: Support complet pour l'Espagnol, le Catalan, l'Anglais et le Français
  • Responsive: Fonctionne sur appareils mobiles et desktop
  • Exportation Améliorée: CSV avec toutes les données de la course
  • Tri Intelligent: Les retirés sont triés automatiquement
  • Persistance Améliorée: Sauvegarde automatique dans multiples situations
