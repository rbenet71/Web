Ayuda Cronómetro Control 80%

Descripción General

El Cronómetro Control 80% es una herramienta diseñada para carreras de BTT y Ciclo-cross que implementa la regla del 80% para determinar cuándo un corredor debe ser retirado de la competición.

Funcionalidades Principales

Gestión de Carreras

  • Nueva Carrera: Crear una nueva carrera con nombre personalizado
  • Seleccionar Carrera: Cambiar entre diferentes carreras guardadas
  • Eliminar Carrera: Borrar una carrera existente

Control del Cronómetro

  • Iniciar/Parar: Controlar el cronómetro principal
  • Fijar Tiempo: Establecer manualmente la hora de salida o el tiempo actual
  • Reiniciar: Restablecer completamente el cronómetro y todos los datos

Registro de Vueltas

  • Paso Primer Corredor: Registrar el tiempo de cada vuelta del líder
  • Vuelta Actual: Muestra el número de vuelta en curso

Control de Corredores

  • Retirar Corredor: Marcar corredores como retirados en la vuelta actual
  • DNF-Retirado: Marcar corredores como No Finalizaron (vuelta 0)

Cómo Usar

1 Configurar la Carrera

Crear una nueva carrera o seleccionar una existente. Iniciar el cronómetro cuando comience la carrera.

2 Registrar la Primera Vuelta

Presionar "Paso Primer Corredor" cuando el líder complete la primera vuelta. Este tiempo establece la base para el cálculo del 80%.

3 Seguimiento Continuo

Registrar cada vuelta subsiguiente del líder. El sistema calcula automáticamente el tiempo límite para cada vuelta.

4 Control de Retirados

Usar "Retirar Corredor" para corredores que abandonan durante la carrera. Usar "DNF-Retirado" para corredores que no finalizan.

5 Exportar Resultados

Al finalizar, exportar los datos para análisis posterior.

Regla del 80%

El sistema calcula el tiempo límite para cada vuelta sumando:

  • Tiempo de la última vuelta registrada del líder
  • 80% del tiempo de la primera vuelta

Cuando un corredor supera este tiempo límite, debe ser retirado de la competición.

Características Avanzadas

  • Edición de Tiempos: Modificar tiempos de vueltas y corredores retirados
  • Idiomas: Soporte para Español, Catalán e Inglés
  • Persistencia: Los datos se guardan automáticamente
  • Responsive: Funciona en dispositivos móviles y desktop

Atajos

  • Doble clic en el display del cronómetro: Abrir modal para fijar tiempo
  • Icono de ayuda: Acceso rápido a este documento de ayuda

Instalación como App en el Móvil

Para iPhone/iOS:

  1. Abrir Safari y navegar a la página del cronómetro
  2. Tocar el icono de compartir (cuadrado con flecha hacia arriba)
  3. Desplazarse hacia abajo y seleccionar "Añadir a pantalla de inicio"
  4. Personalizar el nombre si se desea y tocar "Añadir"
  5. La app aparecerá en la pantalla de inicio como una aplicación nativa

Para Android:

  1. Abrir Chrome y navegar a la página del cronómetro
  2. Tocar el menú de tres puntos en la esquina superior derecha
  3. Seleccionar "Añadir a pantalla de inicio"
  4. Personalizar el nombre si se desea y tocar "Añadir"
  5. La app aparecerá en la pantalla de inicio como una aplicación nativa

Ventajas de instalarla como App:

  • Acceso rápido directo desde la pantalla de inicio
  • Funciona sin conexión a internet después de la primera carga
  • Interfaz similar a una aplicación nativa
  • No requiere descarga desde tiendas de aplicaciones

General Description

The Stopwatch 80% Control is a tool designed for MTB and Cyclo-cross races that implements the 80% rule to determine when a rider should be withdrawn from the competition.

Main Features

Race Management

  • New Race: Create a new race with custom name
  • Select Race: Switch between different saved races
  • Delete Race: Delete an existing race

Stopwatch Control

  • Start/Stop: Control the main stopwatch
  • Set Time: Manually set start time or current time
  • Reset: Completely reset the stopwatch and all data

Lap Recording

  • First Rider Pass: Record the time of each lap by the leader
  • Current Lap: Shows the current lap number

Rider Control

  • Withdraw Rider: Mark riders as withdrawn in the current lap
  • DNF-Withdrawn: Mark riders as Did Not Finish (lap 0)

How to Use

1 Set Up the Race

Create a new race or select an existing one. Start the stopwatch when the race begins.

2 Record First Lap

Press "First Rider Pass" when the leader completes the first lap. This time sets the base for the 80% calculation.

3 Continuous Tracking

Record each subsequent lap by the leader. The system automatically calculates the cutoff time for each lap.

4 Withdrawn Riders Control

Use "Withdraw Rider" for riders who abandon during the race. Use "DNF-Withdrawn" for riders who do not finish.

5 Export Results

When finished, export data for later analysis.

80% Rule

The system calculates the cutoff time for each lap by adding:

  • Time of the last recorded lap by the leader
  • 80% of the first lap time

When a rider exceeds this cutoff time, they must be withdrawn from the competition.

Advanced Features

  • Time Editing: Modify lap times and withdrawn riders' times
  • Languages: Support for Spanish, Catalan and English
  • Persistence: Data is automatically saved
  • Responsive: Works on mobile devices and desktop

Shortcuts

  • Double click on the stopwatch display: Open modal to set time
  • Help icon: Quick access to this help document

Installation as Mobile App

For iPhone/iOS:

  1. Open Safari and navigate to the stopwatch page
  2. Tap the share icon (square with upward arrow)
  3. Scroll down and select "Add to Home Screen"
  4. Customize the name if desired and tap "Add"
  5. The app will appear on the home screen like a native app

For Android:

  1. Open Chrome and navigate to the stopwatch page
  2. Tap the three-dot menu in the upper right corner
  3. Select "Add to Home Screen"
  4. Customize the name if desired and tap "Add"
  5. The app will appear on the home screen like a native app

Advantages of Installing as App:

  • Quick access directly from home screen
  • Works offline after first load
  • Interface similar to native app
  • No download required from app stores
